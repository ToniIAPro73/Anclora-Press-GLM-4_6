## 5. APLICACIÓN DE DEGRADADOS

### Ubicaciones Recomendadas

#### Botones
- **Botones Primarios:** Degradado Navy (hover más pronunciado)
- **Botones Secundarios:** Degradado Gold (hover suave)
- **Botones Acentos:** Degradado Teal (hover moderado)
- **Transición:** 200ms ease-in-out

#### Fondos y Superficies
- **Header:** Degradado Navy sutil
- **Sidebar:** Degradado Fondo muy sutil
- **Cards Principales:** Sin degradado (fondo sólido)
- **Secciones Destacadas:** Degradado Gold/Teal según contexto

#### Elementos Decorativos
- **Líneas Divisoras:** Degradado horizontal (fade in/out)
- **Bordes de Tarjetas:** Degradado vertical sutil
- **Fondos de Secciones:** Degradado muy ligero para profundidad

```

Texto sobre Gold:       Navy o Blanco (4.5:1)
Texto sobre Aqua:       Navy o Blanco (4.5:1)
Texto sobre Teal:       Blanco o Sand (4.5:1)
```

### Modo Oscuro
- Contraste mínimo 4.5:1 para texto pequeño
- Contraste mínimo 3:1 para texto grande
- Todos los botones claramente distinguibles
- Iconos con suficiente contraste

## 12. NOTAS IMPORTANTES

### Elegancia y Armonía
- Todos los degradados usan ángulo 135° (diagonal sutil)
- Variación de color máxima: 10-15% para mantener sutileza
- Transiciones suaves en modo hover (200-300ms)
- Consistencia visual entre temas claro y oscuro

### Preservación de Funcionalidad
- Todos los módulos existentes se mantienen
- Cambios principalmente visuales (CSS/colores)
- API de importación mejorada pero compatible
- Estructura de componentes preservada

### Rendimiento
- Degradados CSS nativos (sin imágenes)
- Sin impacto en rendimiento
- Optimizado para todos los navegadores modernos
- Soporte para modo oscuro automático
